DETALLE UX // LÁMINA 3

Módulos Críticos de Decisión Nacional

🧑‍💻 Interfaz Ciudadana

Integración con la API de la Red Nacional de Estaciones Hidrometeorológicas del SENAMHI.

  • 📍 Geolocalización y enlace a estaciones Convencionales / Automáticas.
  • ☀️ Indexación automática del Semáforo de Radiación Solar Ultravioleta (UV).
  • 📊 Monitoreo de calidad del aire en tiempo real para Lima Metropolitana y Callao.

🛡️ Alertas para Gestión de Riesgos

Herramientas para tomadores de decisiones de INDECI, COEN y Gobiernos Regionales.

  • 🌊 Conexión con Servidor Hidrológico (PHISIS) para umbrales de desborde de ríos.
  • ⚠️ Avisos de Niveles de Alerta (Amarillo, Naranja, Rojo) automatizados por distrito.
  • 🌋 Modelamiento numérico complementario de Dispersión de Ceniza Volcánica e Incendios.
DETALLE UX // LÁMINA 4

Sectores de Desarrollo Socioeconómico

🚜 Inteligencia Agroclimática

Monitoreo y herramientas preventivas para la seguridad alimentaria en el campo peruano.

  • ❄️ Predicción de Heladas Agrometeorológicas (Sierras) y Friajes (Selva).
  • 🌾 Acceso directo a Boletines Decadiarios y Mensuales sectorizados del SENAMHI.
  • 📱 Interoperabilidad nativa de datos con la aplicación móvil institucional.

✈️ Corredor Turístico Seguro

Información en tiempo real orientada a la protección de operadores y visitantes extranjeros.

  • ⛰️ Pronósticos puntuales para Machu Picchu, Huaraz y playas del litoral.
  • 💨 Alertas de ráfagas intensas de viento, niebla densa y oleajes anómalos costeros.
  • 🗺️ Capas GIS con rutas viales sugeridas según la criticidad meteorológica actual.
